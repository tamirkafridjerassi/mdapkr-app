# mdapkr — FastAPI + MongoDB Microservice

## 🚀 Overview

This is a backend API for managing:
- Crew members (`/person`)
- Missions (`/mission`)

Built with:
- FastAPI
- MongoDB
- Docker + docker-compose

## 🛠️ Run Locally

```bash
cd fastapi-app
docker-compose up --build
```

Swagger UI: [http://localhost:8000/docs](http://localhost:8000/docs)
